패치 1 — 학습자료 확장

배포일: 2026년 7월 15일

기존 Section 0~5를 보완하고, 이후 학습을 위한 Section 6~12와 실습 파일을 추가했습니다.

기존 Section 0~5 변경사항

  • API key와 모델 설정 방식을 Section별로 통일했습니다.
  • Section 3의 JSON 응답을 Structured Output으로 변경해 필드 누락으로 인한 오류를 줄였습니다.
  • Section 4의 Workflow 출력과 상태 검증을 보완했습니다.
  • 기존 Notebook에 저장되어 있던 실행 결과를 정리했습니다.
  • 학습 내용과 진행 순서는 변경하지 않았으므로 완료한 Section을 다시 진행할 필요는 없습니다.

사이트와 실습 파일

새로 추가된 Section

  • Section 6: 문서를 검색 가능한 Chunk로 나누고 출처 정보를 유지합니다.
  • Section 7: Embedding과 VectorStore를 이용한 의미 검색을 학습합니다.
  • Section 8: 평가 질문을 이용해 Retriever의 검색 성능을 확인합니다.
  • Section 9: 검색 근거와 인용을 검증하는 Vector RAG를 구성합니다.
  • Section 10: Retrieval과 Generation을 나누어 평가하고 개선합니다.
  • Section 11: Tool을 사용하는 Agent와 실행 흐름 제어를 학습합니다.
  • Section 12: 앞에서 학습한 내용을 연결해 미니 프로젝트를 진행합니다.

사이트 및 파일 변경사항

  • 메인 학습 지도와 Section별 페이지에 Section 6~12를 추가했습니다.
  • 다운로드 ZIP에 Notebook 00~12와 새 실습 데이터를 포함했습니다.
  • 각 Notebook 상단에 API key 필요 여부를 표시하고, 필요한 파일에는 입력 셀을 추가했습니다.
  • Section 페이지에 이전·다음 이동과 참고자료 안내를 추가하고 모바일 화면을 정리했습니다.

검증

  • 최신 ZIP을 새 폴더에 풀어 Notebook 00~12를 순서대로 실행했습니다.
  • API가 필요한 실습은 수업용 프로젝트 key로 실제 호출해 확인했습니다.
  • 구조화 출력, 상태 분기, 검색 근거와 인용 검증을 확인했습니다.
  • 사이트 링크, 모바일 화면, 다운로드 파일과 공개 파일의 개인정보·API key 포함 여부를 점검했습니다.

기존 자료를 내려받아 사용하고 있다면 Section 6을 시작하기 전에 최신 ZIP을 다시 받아주세요. API key는 ZIP에 포함되지 않으므로 필요한 Notebook의 상단 셀에 다시 입력해야 합니다.
